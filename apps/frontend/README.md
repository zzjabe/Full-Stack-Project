## Project Description
The Library Management System is a web application
designed to help users browse, organize, and 
manage library resources. This application will provide 
an intuitive interface for viewing books, managing personal 
lists, and tracking borrowing activity.

Moreover,The project is being developed using an Agile methodology
which is  sprint-based approach and Development begins with a React 
and TypeScript frontend, with additional features and backend integration
planned in future sprints.

## High-Level User Stories
- As a user, I want to browse a catalog of books so that I can easily 
find items available in the library.
- As a user, I want to log in and manage my borrowed books so that
 I can keep track of what I have checked out.
- As a user, I want to save books to a personal list so that
 I can quickly access them later.

## Technology Stack
- Frontend: React with TypeScript (Vite)
- Version Control: Git and GitHub
- Project Management: GitHub Projects (Kanban)
